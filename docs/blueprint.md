# **App Name**: My Field Planner

## Core Features:

- Today's Tasks Dashboard: A dynamic home dashboard displaying tasks due today, with overdue tasks highlighted. Users can mark tasks as complete with immediate visual feedback.
- New Task Entry: A clean, portrait-mode form for adding new tasks, including client name, reason, date, selected area (from RouteMaster), and priority.
- Area Route Builder: A visual, node-based or nested-list interface for users to build and manage their hierarchical geographical route network, starting from 'Home'.
- Dynamic Field Plan Generation: Upon selecting a destination, the system traverses the RouteMaster tree backwards to identify passing villages and intelligently filters associated tasks, splitting them into 'Today's Route Tasks' and 'Upcoming Route Tasks'.
- Comprehensive Task Views: Access to 'All Task View' with fuzzy search across various fields for incomplete tasks, and a dedicated 'History' view for all completed tasks.
- Mobile-Optimized Navigation: A persistent top app bar with a hamburger menu for global navigation, and a floating action button on the home page for quick task creation or field plan initiation.

## Style Guidelines:

- Primary color: A deep, professional blue (#294A99), selected to convey reliability and focus, contrasting clearly with light backgrounds.
- Background color: A very light, desaturated blue-gray (#ECF1F7), offering a clean and calm canvas that harmonizes with the primary color.
- Accent color: A vibrant indigo (#611EDD), chosen for highlighting important actions and elements, providing a distinct yet complementary contrast to the primary.
- Primary font: 'Inter' (sans-serif), chosen for its modern, legible, and neutral aesthetic, suitable for both headlines and body text to ensure clarity and professionalism.
- Utilize a consistent set of clean, minimalist line icons that align with material design principles, promoting intuitive understanding and reducing visual clutter for one-handed mobile use.
- Prioritize a mobile-first, responsive layout with generous touch targets, bottom sheets for transient actions, and distinct sections for data organization, ensuring efficient single-handed navigation and data entry.
- Implement subtle, fluid animations for UI transitions, such as modal openings, row deletions with fade-out effects, and bottom-sheet sliding, to enhance user feedback and overall experience without distracting from core tasks.
